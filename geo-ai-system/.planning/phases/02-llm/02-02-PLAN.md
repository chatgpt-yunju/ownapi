  - Get access_token via getAccessToken
  - Build ErnieRequest with model=config.modelId, messages, temperature=0.7, max_output_tokens=2000
  - POST to "https://aip.baidubce.com/rpc/2.0/ai_custom/v1/wenxinworkshop/chat/completions?access_token=" + access_token

  - Return result

- Override countTokens: Return content.length() / 4
