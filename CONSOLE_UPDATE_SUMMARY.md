- 📊 **仪表盘** - 显示余额、今日调用、tokens、费用
- 📊 **数据统计** - 详细统计数据和图表（带刷新按钮）

- 🔑 **API Keys** - 管理 API 密钥
- 📝 **调用日志** - 查看调用历史

#### 邀请奖励
- 🎁 **邀请码** - 生成和分享邀请码
- 💰 **我的奖励** - 查看奖励记录

#### 其他
- 🔔 **消息通知** - 系统通知中心
- 🧪 **API 调试** - 在线测试 API

## 功能特点

### 1. 数据统计页面
- 总调用次数、总 Tokens、总费用、平均费用
- 模型使用分布图表
- 近30天趋势图表
- 刷新按钮

### 2. 加油包页面
- 4种充值选项（¥50/¥100/¥500/¥1000）
- 大额充值赠送额外余额
- 一键购买，支付宝支付

### 3. 邀请码页面
- 显示个人邀请码
- 一键复制功能
- 邀请统计（人数、累计奖励、待发放）
- 邀请记录列表

### 4. 我的奖励页面
- 累计奖励、已领取、待领取
- 奖励记录详情

### 5. 消息通知页面
- 显示所有通知
- 标记已读/未读
- 一键全部标记为已读
